# Enhanced To-Do List

This is an Enhanced To-Do list application built with HTML, CSS, and JavaScript. It supports adding, completing, deleting, searching, and saving tasks using localStorage.

# Features
# Add Tasks
Users can add tasks to the list.

# Unique ID
Each task is stored as an object:
```json
{
    "id": 123456,
    "text": "Buy groceries",
    "completed": false
}
